Le principe est un Billard en plus simplifié, il faut juste rentrer toutes les balles avant de rentrer la balle blanche.

Si la balle blanche est rentrer avant les autres elle peut être replacée n'importe où sur la table, à l'exeption de l'emplacement des autres balles.

Il y a une petite aide en maintenant la touche 'A'. Avec l'affichage des règles de jeu, la possibilité d'arrêter toutes les balles et celle de recommencer la partie.

Pour le tir il y a une prévisualisation, lors du tir il n'y a plus la possibilité de tirer tant que la balle blanche n'est pas immobile.

Les collisions avec les bords sont présentes et aussi celles des trous dans lesquelles doivent rentrer toutes les balles.

Lors d'un choc entre deux balles, le calcul des vitesses est réalisé et il est réalisé avec une matrice transposée pour recalculer les nouvelles vitesses.

Si une balle est rentrée, alors son état change. L'état des ballles sont affichés en haut à gauche. Vide pour la balle sur la table et pleine pour la balle qui est dans le trou.

Une fois toutes les balles rentrées, l'écran des statiqtiques apparaît et vous montre différentes statiqtiques sur vos balles (temps de survie, vitesse moyenne, nombre de rebond sur la bande, nombre de collision avec une autre balle, nombre de fois où les balles sont rentrées, nombre d'ârret manuel des balles).

Si vous passez votre souris sur les balles qui sont en haut vous avez la possibilité de voir les statistiques individuelles de chacunes d'entre elles.

Et puis pour terminer vous n'avez plus qu'à faire 'esc' pour fermer le programme.
